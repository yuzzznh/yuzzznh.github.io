1. Number Classification
- **rational number = 유리수** <-> irrational number = 무리수
- whole number = 범 자연수 (0 및 자연수)
- 0은 even=짝수 에 포함된다
- prime number = 소수 (약수 2개) / composite number = 합성수 / 1은 소수가 아니다! 약수가 1개뿐이니까.
- **prime factor** = 소인수! 소수인 약수. (factor = 약수)
  
1. Special Number
- consecutive integer = 연속하는 정수 (1씩 차이나는.)
- consecutive even / odd integer = 연속하는 짝수/홀수 (2씩 차이나는.)
  
1. Multiples
- multiples = 배수 === **divisible** = 나눠떨어지는 수!!!
- factor = 약수 === **[divisors](.)** = 나눌 수 있는 수!!!
- **[quotient](.)** = 몫 === **[answer](.)** = 답 아니고 몫!!!
- **[dividend](.)** = 피제수. 나눠지고 있는 수. (12 / 4 = 3에서 4는 factor이자 divisor, 12는 피제수)
- remainder = 나머지

1. Arithmetic Expression
- 더하기
  - add to, sum, total
  - **A more than B** - B를 기준으로! A가 더해진다. B+A.
- 빼기
  - subtract from
  - **A less than B** - B를 기준으로! A가 적다. B-A. than을 기준으로 뒤에 나온 게 주체!
  - A exceed B by C -> A는 B를 C만큼 초과한다 -> A - B = C
    - by는 단독으로 나오면 곱하기가 되지만, exceed와 짝꿍으로 나올 땐 등호(=) 역할을 한다!
- 곱하기
  - multiply, **by**(단독), times
  - **product of A and B** = A x B
- 나누기
  - divided by
  - **[A per B](.)** === A **over** B ===** A / B**. 앞에있는거 나누기 뒤에있는거!

1. Fraction / Decimal
- digit = 자리수
  - 12는 2 digit number
- **[numerator](.) = 분자!!!!!**
- **[denominator](.) = 분모!!!!!**
- **2/3** - 한국어로는 분모부터 읽어서 "삼분의 이" 라고 읽지만, 영어로는 분자부터 읽어서 "**2 over 3**" 라고 읽는다
  - 그 안에서 기수와 서수를 섞어서 쓴다. 분자는 기수를, 분모는 서수를 쓴다.
  - 2/3 = **two thirds**!!!!!!!
- round to, nearest to = 반올림

1. Raical / Exponent
- square = 제곱 <-> square root of ~ = 제곱근 (코딩할 때 sqrt!)
- **cube** = 세제곱 <-> **cube root **of ~ = 세제곱근!

1. Function
- **quadrant** = 사분면
  - 1st, 2nd, 3rd, 4rd quadrant가 각각 1, 2, 3, 4사분면이다. (시계 반대방향!)
- origin = 원점!!!
- x-axis = x축 -> **x-intercept** = x절편
- y-axis = y축 -> **y-intercept** = y절편
- **slope = 기울기!!!**
- parallel line = 평행선
- **[perpendicular](.)** line = (교차하는 선에 대한) 수직선 **=/=** vertical line = (지면/x축에 대한) 수선
- vertex/vertices = 꼭지점

1. Triangle
- **equilateral** triangle = 정삼각형 (equilateral은 변의 길이가 같다는 뜻. 정다각형을 의미)
- **[isosceles](.)** triangle = 이등변삼각형
- **right** triangle = 직각삼각형!!!
  - leg = 직각삼각형의 밑변과 높이를 각각 나타낸다 (둘이 구별 X) <-> **[hypotenuse](.)** = 빗변
    - 사실 leg라고 했기때문에!!! 직각삼각형이 되는 것.
      - A triangle has 2 **leg** size of X and Y respectively **=>** 그 삼각형은 **직각삼각형**!!!

1. Quadrilateral (사각형)
- parallelogram = 평행사변형!!!
  - 주의) 평행사변형의 대각선은, 면적은 이등분하지만, 각을 이등분하는건 아니다!!!
- diagonal = 대각선
- rectangle = **직**사각형
- square = **정**사각형 (제곱도 정사각형도 square~)
- **[rhombus](.) = 롬버스는 마름모!!**

1.  Circle (원)
- diameter = 지름 <-> radius = 반지름
- [chord('코드')](.) = 현: 원 안의 선분! <-> [arc](.) = 호: 곡선! $$$2\pi r$$ 관련. <-> [sector](.) = 부채꼴. $$\pi r^2$$ 관련.
- 지름과 현의 관계: chord에서 가장 긴 길이는 diameter (원의 center를 지날 때)

1.  Set (집합)
- either A or B = 합집합 = A 또는 B (교집합 아님!)
- both A and B = 교집합
- neither A nor B = 여집합
- [either A or B, **but not both**](.) = 차집합 = **합집합 - 교집합** = A + B - **2** A ∩ B

1.  Statistics
- **average** = **mean** = arithmetic mean = **평균** (산술평균) (합산!)
- **median** = 중앙값 (**위치**상 가운데! medium이랑 관련지어서 기억하자)
- **[mode](.)** = 최빈값 (가장 빈번하게 나오는 숫자! frequency 가장 높음 -> **영향력**!)
- **[range](.) = 범위 = max값 - min값** -> min ~ max=min+range 
- normal distribution = 정규분포
  - 표준정규분포가 아님에 유의하자! standard 아님!
  - 여기선 mean, median, mode 세 값이 다 같다! -> 대칭분포도라고 부르기도 한다!
- **[percentile](.)** = 백분위
  - 낮은점수부터 앞쪽으로 세준 것.
  - GRE도 percentile로 뜬다. *높을수록 잘 친 것*. 99%가 1등한 것!
