| 기능                 | 마크다운 형식                                       | 설명                                                                               |
| -------------------- | --------------------------------------------------- | ---------------------------------------------------------------------------------- |
| 제목                 | `# 제목`                                            | `#` 뒤에 스페이스를 누르면 제목이 되고, 여러 개를 붙일수록 제목 크기가 작아집니다. |
| 이탤릭체             | `*이탤릭*`                                          | 글자를 이탤릭체로 만듭니다.                                                        |
| 글자 진하게          | `**진하게**`                                        | 글자를 진하게 만듭니다.                                                            |
| 취소선               | `~~취소선~~`                                        | 글자에 취소선을 긋습니다.                                                          |
| 하이라이트           | `==하이라이트==`                                    | 글자를 하이라이트합니다.                                                           |
| 태그                 | `#태그`                                             | 스페이스 없이 `#` 뒤에 붙이면 태그가 됩니다.                                       |
| 글머리기호           | `- 글머리기호`                                      | 목록을 만듭니다.                                                                   |
| 문단번호 매기기      | `1. 문단번호`                                       | 번호가 매겨진 목록을 만듭니다.                                                     |
| 인용                 | `> 인용`                                            | 인용문을 만듭니다.                                                                 |
| 주석                 | `%%주석%%`                                          | 주석을 만듭니다.                                                                   |
| 체크박스             | `- [ ] 체크박스`                                    | 체크박스를 만듭니다.                                                               |
| 이미지 삽입          | `![[파일이름.확장자]]`                              | 이미지를 삽입합니다. (이미지 파일은 옵시디언 폴더에 있어야 합니다)                 |
| 문서 링크            | `[[문서이름]]`                                      | 다른 문서로의 링크를 만듭니다.                                                     |
| 링크 이름 변경       | `[[문서이름\|원하는말]]`                            | 링크 텍스트를 원하는 말로 변경합니다.                                              |
| 문서 특정 부분 링크  | `[[문서이름#제목]]`, `[[문서이름^문장]]`            | 문서의 특정 부분으로 링크를 만듭니다.                                              |
| 하이퍼링크 이름 변경 | `[보이는 문장](사이트주소)`                         | 하이퍼링크의 표시 텍스트를 변경합니다.                                             |
| 구분선               | `***`                                               | 구분선을 만듭니다.                                                                 |
| 메타데이터           | `---메타데이터---`                                  | 메타데이터를 추가합니다.                                                           |
| 표                   | 하단 참고       | 표를 만듭니다. |
| 인라인 코드          | `` `코드` ``                                        | 인라인 코드를 표시합니다.                                                          |
| 블록 코드            | \````\n코드\n\````                                  | 여러 줄의 코드를 블록으로 표시합니다.                                              |
| 수평선               | `---`                                               | 수평선을 만듭니다.                                                                 |
| 강조                 | `_강조_`                                            | 글자를 강조합니다. (이탤릭체와 비슷)                                               |
| 링크                 | `[링크 텍스트](URL)`                                | 하이퍼링크를 만듭니다.                                                             |
| 이미지               | `![이미지 설명](이미지 URL)`                        | URL로 이미지를 삽입합니다.                                                         |
| 인라인 수학식        | `$수학식$`                                          | 인라인 수학식을 표시합니다. (일부 마크다운 엔진에서 지원)                          |
| 블록 수학식          | `$$\n수학식\n$$`                                    | 여러 줄의 수학식을 블록으로 표시합니다. (일부 마크다운 엔진에서 지원)              |

| ----- | ----- |
